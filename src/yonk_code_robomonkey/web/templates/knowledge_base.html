About the Knowledge Base

The knowledge base stores and indexes documentation for RAG (Retrieval Augmented Generation) queries.

  • PDF Documents - Extracted with structure preservation (headings, tables, code blocks)
  • Markdown/HTML - Parsed with heading hierarchy and code block detection
  • Smart Chunking - 500-1500 token chunks with overlap for context continuity
  • Auto-Tagging - Oracle constructs (ROWNUM, CONNECT BY, DECODE) and EPAS features are auto-detected
  • Hybrid Search - Combines vector similarity (60%) and full-text search (40%)

Use MCP tools doc_search and doc_get_context for programmatic access.
