About the Knowledge Base

The knowledge base stores and indexes documentation for RAG (Retrieval Augmented Generation) queries.

Features

  • Ask Docs (RAG Q&A) - Ask natural language questions and get AI-synthesized answers with citations
  • Hybrid Search - Combines vector similarity (60%) and full-text search (40%)
  • Smart Chunking - 500-1500 token chunks with overlap for context continuity
  • Auto-Tagging - Oracle constructs and EPAS features are auto-detected

Ask Docs vs Search

  • Ask Docs - Get ONE synthesized answer from multiple sources with inline citations [1], [2]
  • Search - Browse ranked chunks with scores and optional per-chunk summaries

Use MCP tools kb_ask, kb_search, and kb_get_context for programmatic access.
